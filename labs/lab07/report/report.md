листинга в командной строке. Создайте файл листинга для программы из
файла lab7-2.asm

![Файл листинга lab7-2](image/09.png){ #fig:009 width=70%, height=70% }

строка 128

* 128 - номер строки

* 0000009C - адрес

* 53 - машинный код

* push ebx - код программы

строка 129

* 129 - номер строки

* 0000009D - адрес

* 51 - машинный код

* push ecx- код программы

строка 130

* 130 - номер строки

* 0000009E - адрес

* 52 - машинный код

* push edx - код программы
